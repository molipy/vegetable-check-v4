# vegetable-check-v4
基于深度学习的【蔬菜识别】系统实现~2026原创+计算机毕设

## 项目介绍

本项目是一个基于深度学习的常见蔬菜图像识别Web系统。系统旨在通过计算机视觉技术，实现对土豆、大白菜、大葱、莲藕、菠菜、西红柿、韭菜、黄瓜这8种日常常见蔬菜的自动、快速、准确分类与识别。项目在技术实现上，前端采用Vue3结合Element Plus组件库构建现代化且响应迅速的用户界面，提供图片上传、结果展示、用户登录交互等功能；后端基于Python的轻量级Web框架Flask进行开发，结合SQLite数据库和JWT认证机制，提供稳定可靠的RESTful API接口服务与数据管理；核心算法部分采用TensorFlow框架，并引入了经典的ResNet50深度残差网络模型进行图像特征提取与分类。通过迁移学习技术，大幅提升了模型在特定蔬菜种类识别上的准确率和泛化能力。
